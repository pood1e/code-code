# Agent Workbench

Monorepo scaffold for a personal agent configuration workbench.

## Packages

```text
packages/
  backend/   NestJS service scaffold
  frontend/  React + Vite app scaffold
  shared/    Shared TypeScript package scaffold
```

## Getting Started

```bash
pnpm install
cp packages/backend/.env.example packages/backend/.env
pnpm dev
```

## Commands

```bash
pnpm build
pnpm lint
pnpm format
```
